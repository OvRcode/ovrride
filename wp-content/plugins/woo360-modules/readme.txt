== Description ==

Various modules and tweaks for the Woo360 Builder

A few notes about the sections above:

*   Module 1 - Woo Commerce Login/Cart Info Links

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. View the modules in your front end builder interface in the drop down section for modules, named Woo360 Modules
