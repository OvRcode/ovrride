 Remove the Qty from the product page in Woocommerce. according to Product Type.

== Description ==

Remove the Qty from the product page in Woocommerce. it has an option page where you can set which to Product Type should not show qty textbox

== Installation ==

1. Upload `woo-remove-qty-text` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go To setting -> Woocommerce Remove Qty option.
4. select the product type where you want to hide quantity.
5. Plugin is ready for work.
