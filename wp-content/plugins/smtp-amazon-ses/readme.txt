== Description ==
**Send WordPress emails through Amazon SES server using YaySMTP**

Amazon Simple Email Service (SES) is a versatile mail service provider. It is also the best SMTP server with great email deliverability. If you need a simple way to integrate **Amazon SES** with WordPress, YaySMTP is the **best** choice.

It costs nothing to connect with AWS / Amazon SES.

YaySMTP for WP SES comes without baffling configuration options. Any WordPress beginners can set it up successfully. 

Moreover, this WP SES plugin provides advanced options that you can't find in other Amazon SES SMTP plugins on WordPress search results.

- **Easy setup**: clear interface and simple user experience 
- **Email log**: set time frame for auto-delete, show/hide columns
- **Email actions**: display, search, view, delete emails easily
- **Send test email**: one click to send email for testing
- **Compact UI**: do everything without leaving the page or browser tab

Absolutely! YaySMTP doesn't collect or store any personal information. So rest assured.

Yes, you can use it free of charge. YayCommerce WP SMTP plugin for Amazon SES comes with all features that you can find in the description above.

You can customize WooCommerce email templates by using [YayMail](https://wordpress.org/plugins/yaymail/) free plugin. Fully compatible and optimized for WooCommerce.

== Installation ==
1. Upload the entire plugin folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
